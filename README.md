# Getting Started
These instructions tell the prerequisites and show how to run the project on your local machine.

## Prerequisites and Installation

**Node**
  
 - Install Node: https://nodejs.org/en/
 - To install the dependencies, run: 

> npm install

## Running

 - Run: 

> npm start

## Tests

- Run:

> npm test
